# 文件处理工具

这是一个简单的文件处理工具，支持上传 JS 文件并进行处理。

## 功能特点

- 支持拖拽上传文件
- 支持 JS 文件格式
- 实时显示处理结果
- 支持保存处理结果为 TXT 文件

## 使用方法

1. 点击上传区域或拖拽文件到上传区域
2. 选择 JS 文件
3. 点击"处理文件"按钮
4. 等待处理完成
5. 查看处理结果
6. 可选择保存结果为 TXT 文件

## 开发环境设置

1. 克隆仓库
```bash
git clone https://github.com/YOUR_USERNAME/file-processor.git
```

2. 修改 `config.js` 文件中的 API 地址

3. 使用本地服务器运行项目
```bash
# 如果使用 Python
python -m http.server 8000

# 如果使用 Node.js
npx serve
```
